# About

To run the application in your localhost, it's better to have a server running

## Running

    $ cd /my/path/to/project/

    $ python -m SimpleHTTPServer 8080

or 

    $ python -m http.server 8080

then in your browser:
    
    $ http://127.0.0.1:8080/
